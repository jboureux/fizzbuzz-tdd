# FizzBuzz TDD
##  Installation
After cloning the project, initialize a python virtual environnement using virtualenv into the folder

```shell
pip3 install virtualenv
python3 -m virtualenv .venv
```

Then, enter into the virtual env shell by executing the activate script in the .venv folder.

On Linux use this command:
```shell
source .venv/bin/activate 
```

Once you're in the shell, intall pytest
```shell
pip3 install pytest
```
